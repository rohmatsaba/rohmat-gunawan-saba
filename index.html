HANYA SEORANG ANAK LAKI-LAKI YANG PENDIAM

Ditulis Oleh: Rohmat Gunawan Sabapada tanggal 20 juni 2025

Saya mendapat tugas untuk membuat tampilan website sederhana pakai HTML dan CSS. Di tugas ini, saya belajar menyusun bagian-bagian dasar dari sebuah halaman web, seperti judul (header), menu, isi konten, gambar, dan bagian bawah (footer). Semua itu saya atur pakai HTML, supaya strukturnya jelas dan rapi. Saya juga menambahkan elemen lain seperti link, biar isi websitenya lebih lengkap.

Setelah bagian dasarnya selesai, saya percantik tampilannya pakai CSS. Di sini saya atur warna, ukuran tulisan, jenis font, jarak antar elemen, dan posisi kontennya. Saya juga coba-coba kasih efek supaya websitenya lebih menarik saat dilihat. Lewat tugas ini, saya jadi lebih paham gimana cara membuat website dari awal, sekaligus belajar menggabungkan coding dengan desain biar hasilnya bagus dan enak dilihat.
